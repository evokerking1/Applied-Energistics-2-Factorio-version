# ⚠️ **PONTOS QUE PODEM DAR PROBLEMA**

## 1. 🔓 Unlock muito “em lote”

Aqui:

```lua
{
    "certus-quartz-crystal",
    "certus-quartz-dust",
    "damaged-budding-certus-quartz",
    "chipped-budding-certus-quartz",
    "flawed-budding-certus-quartz",
    "certus-quartz-block"
}
```

👉 Você libera **muita coisa de uma vez**

💡 Problema:

* jogador não entende o que é importante
* perde sensação de progressão

### ✔️ Melhor:

Dividir em micro-techs:

```lua
-- base
certus-quartz-crystal

-- depois
certus-processing (dust)

-- depois
budding-blocks
```

---

## 2. ⚡ Fluix desbloqueado só por craft

```lua
LDA.createTechnologyCraftEntityTrigger(...)
```

👉 Aqui você perde a “mágica” do AE2

💡 No AE2, Fluix é um ritual (água + itens)

### ✔️ Melhor trigger possível:

* jogar itens na água
* ou obter fluix pela primeira vez

Se não tiver isso tecnicamente:
👉 usar **craft + item obtido**

---

## 3. 🔁 Falta de “loop” de dependência

Atualmente é linear:

```
A → B → C → D
```

Mas o ideal é:

```
A → B → C
     ↓
     precisa de A melhorado
```

💡 Exemplo:

* Charger → precisa energia
* energia → precisa novos itens
